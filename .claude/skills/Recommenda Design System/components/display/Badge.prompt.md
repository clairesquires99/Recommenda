Compact pill label for categories, media types ("Book", "Film"), and statuses. Use `uppercase` for eyebrow-style labels.

```jsx
<Badge variant="accent" uppercase>Recommended</Badge>
<Badge variant="solid">Popular</Badge>
<Badge variant="neutral" size="sm">2024</Badge>
```
Variants: `accent` (soft blue), `solid` (filled blue), `neutral`, `outline`, plus `success`/`warning`/`danger` softs.
